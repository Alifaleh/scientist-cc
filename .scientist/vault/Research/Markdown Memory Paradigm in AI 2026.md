# The Markdown Memory Paradigm (2026)

> [!note] Key Insight
> 2026 marks the convergence on markdown files as the standard memory primitive for AI agents. Letta (MemGPT) moved from database to MemFS (markdown). Manus uses markdown as durable memory. Claude Code uses CLAUDE.md + auto-memory. scientist-cc uses an Obsidian vault. The industry is validating: markdown is the right format for AI memory.

## Why Markdown Won
1. **LLM-native:** Models trained on internet text process markdown naturally
2. **Human-inspectable:** Users can read, edit, verify AI memory in any text editor
3. **Version-controllable:** Git tracks changes, enables rollback, preserves history
4. **Tool-compatible:** Every CLI tool (grep, sed, cat) works with markdown
5. **Schema-flexible:** YAML frontmatter provides structure; prose provides context
6. **Zero dependencies:** No database, no API, no infrastructure
