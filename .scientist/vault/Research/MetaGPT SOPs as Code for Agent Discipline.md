# MetaGPT: SOPs as Code for Agent Discipline

> [!note] Key Insight
> MetaGPT encodes Standard Operating Procedures (SOPs) directly into the agent framework, requiring structured artifacts at each stage. This dramatically reduces errors because agents can't skip steps or produce unstructured output. Our loop.md IS an SOP — and our templates (hypothesis, observation, principle) enforce structured artifacts.

## Core Idea
- Each agent role has an SOP defining what artifacts it must produce
- The framework ENFORCES the SOP — outputs must match expected structure
- Multi-agent coordination through artifact handoff, not free-form conversation

## What scientist-cc Already Does
- loop.md = SOP for the R&D cycle (11 steps in fixed order)
- Hypothesis template = structured artifact (Statement, Mechanism, Falsification, Confirmation)
- Observation template = structured artifact (frontmatter, callout, What I Saw, Why It Matters)
- Principle template = structured artifact (Core Principle, Evidence, Boundary Conditions, Connections)

## What We Could Add (Phase 3)
- Enforce that every loop step produces a specific artifact type
- Validate artifact structure before allowing progression to next step
- Standardize frontmatter fields across all note types
