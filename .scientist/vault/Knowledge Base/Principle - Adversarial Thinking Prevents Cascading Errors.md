# Principle: Adversarial Thinking Prevents Cascading Errors

> [!note] Core Principle
> The cheapest error to fix is the one you catch before it enters the vault. Adversarial validation (writing 3 ways a hypothesis could be wrong) is the highest-ROI quality gate because it prevents cascading errors in a densely linked knowledge base.
